Определения


Механическое движение - перемещение тела в пространстве относительно других тел с течением времени

Виды

1) По характеру движения
  • Поступательное движение - движение тела, при котором все его точки движутся одинаково
  • Вращательное движение - движение тела, при котором все его точки движутся по окружности с центрами, лежащими на оси вращения
  • Колебательное движение - движение тела, которое повторяется в 2-ух взаимно противоположных направлениях
2) По виду траектории
  • Прямолинейное движение - движение, траекторией которого является прямая линия
  • Криволинейное движение - движение, траекторией которого является кривая линия
3) По скорости
  • Равномерное - движение, при котором тело за любые равные промежутки времени проходит равные расстояния
  • Неравномерное - движение, при котором тело за любые равные промежутки времени проходит неравные расстояния

Материальная точка - тело, размерами и массой которого в данных условиях задачи можно пренебречь

Траектория - воображаемая линия, вдоль которой движется тело
Пройденный путь - длина траектории

Перемещение тела - вектор, проведённый из начального положения тела в конечное

Тело отсчёта - тело, относительно которого рассматривается движение
Система отсчёта = тело отсчёта + система координат + прибор для измерения времени

Скорость

Скорость - величина, показывающая, как быстро с течением времени меняется положение тела в пространстве относительно других тел

Ускорение

Ускорение - величина, показывающая, как быстро меняется скорость тела
