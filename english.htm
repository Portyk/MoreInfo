Времена глаголов

Active voice
- Present Past Future
Simple V / Vs V2 will + V
Continuous am / is / are + Ving was / were + Ving will be + Ving
Perfect have / has + V3 had + V3 will have + V3
Perfect Continuous have / has been + Ving had been + Ving will have been + Ving

1) Отличительной чертой Passive voice от Active voice является то, что деятель (субъект) неизвестен или неважен, а важно действие этого деятеля, совершаемое над каким-либо лицом или предметом (объект).
Пример :
Elon Musk creates rockets (Present Simple Active)
The rockets are created by Elon Musk (Present Simple Passive)

2) Если мы всё-таки захотели указать деятеля (субъект), то мы должны перед ним добавить предлог by или with. Предлог by показывает, кем совершается действие, а предлог with показывает, чем совершается действие.
Пример :
The house is built by the workers
The door will be opened with this key

Условные предложения

Conditionals
Type Сonditional part Main part
0 if + Present Simple ( V / Vs ) Present Simple ( V / Vs )
1 if + Present Simple ( V / Vs ) Future Simple ( will + V )
2 if + Past Simple ( V2 ) would + V
3 if + Past Perfect ( had + V3 ) would have + V3

Type 0
Type Сonditional part Main part
0 if + Present Simple ( V / Vs ) Present Simple ( V / Vs )
Explanation
  • Природные явления
  • Законы физики
  • Факты
Example If the rocket reaches the second cosmic speed (11,2 km/s), it overcomes the gravity of the Earth

Type 1
Type Сonditional part Main part
1 if + Present Simple ( V / Vs ) Future Simple ( will + V )
Explanation
  • Реальные или очень вероятные события, относящиеся к настоящему или будущему времени
Example If I have time, I will help you

Type 2
Type Сonditional part Main part
2 if + Past Simple ( V2 ) would + V
Explanation
  • Маловероятные или гипотетически возможные события, относящиеся к настоящему или будущему времени
  • В этом типе используется частица бы
    (would - сослагательное наклонение)
Example If I had time, I would help you

Type 3
Type Сonditional part Main part
3 if + Past Perfect ( had + V3 ) would have + V3
Explanation
  • Нереальные события, относящиеся к прошлому времени
  • В этом типе используется частица бы
    (would - сослагательное наклонение)
Example If I had had time, I would have helped you
